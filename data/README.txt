================================================================================
                        DATASET DESCRIPTION
          Motion Deblurring of License Plate Images
================================================================================

Source
------
  Azerbaijan License Plates (Cropped)
  https://www.kaggle.com/datasets/bariscelikw/azerbajian-license-plates-cropped

Description
-----------
  This dataset contains 50 cropped license plate images collected from the
  Kaggle dataset above. The images show real-world Azerbaijani vehicle
  license plates captured under various conditions (lighting, angle, distance).

  These serve as the **ground-truth (original)** images for our motion
  deblurring pipeline. Synthetic motion blur is applied to them programmatically
  during the experiment.

Folder Structure
----------------
  data/
  └── original/          50 input license plate images (.jpg)

  The following folders are generated by running the notebook:

  data/
  ├── blurred/           Synthetically motion-blurred images
  ├── kernels/           Visualizations of the motion blur kernels
  ├── restored_inverse/  Images restored using inverse filtering
  ├── restored_wiener/   Images restored using Wiener filtering
  ├── enhanced/          Post-processed (CLAHE + sharpening) images
  ├── thresholded/       Binarized images (adaptive thresholding)
  ├── comparisons/       Side-by-side comparison panels (Original | Blurred | Inverse | Wiener)
  ├── enhanced_comparisons/  Enhancement pipeline panels (Wiener | CLAHE | Sharpened | Thresholded)
  └── metrics/           CSV files with PSNR and SSIM evaluation results

Image Details
-------------
  Format       : JPEG (.jpg)
  Count        : 50 images
  Naming       : licence_1.jpg ... licence_50.jpg
  Processing   : All images are converted to grayscale and resized to
                 256 x 128 pixels during the pipeline execution.

Blur Parameters
---------------
  Kernel lengths : 5, 10, 15, 20 pixels
  Kernel angles  : 0°, 15°, 30°, 45°
  Total blurred images per original : 16 (4 lengths × 4 angles)
  Total blurred dataset size        : 800 images (50 × 16)
